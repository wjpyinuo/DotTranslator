# DotTranslator Privacy Policy

## Telemetry (Optional)

DotTranslator includes optional anonymous telemetry to help improve the product. Telemetry is OFF by default and must be explicitly enabled in Settings.

### What We Collect

- Anonymous instance ID (not tied to user identity)
- App version and OS info
- Feature usage counts
- Heartbeat pings for active user counting
- Translation stats (char count bucket, latency range)

### What We Do NOT Collect

- Translation text or content
- Personal information (name, email, IP)
- Clipboard or file contents
- API keys or tokens

### How to Opt Out

1. Default: Telemetry is OFF unless you explicitly enable it
2. Settings: Toggle in Settings > Privacy
3. Code: PRIVACY_BLOCKED_KEYS enforces data minimization

### Data Retention

- Server-side aggregated data: 90 days
- Raw events: 30 days then auto-archived
- Instance records: Deleted after 180 days inactivity
