## 核心卖点

- **多引擎并行对比**：同一段文本，5 家翻译引擎同时翻译，结果一目了然
- **免费为主**：普通模式 5 家平台免费额度合计 900 万+ 字符/月
- **AI 翻译模式**：支持 DeepSeek-V3、通义千问、Kimi-128k 大模型翻译
- **TTS 语音朗读**：讯飞 TTS + Edge TTS，额度耗尽自动切换
- **Neumorphism 设计**：新拟态 UI，浅色/深色双主题
